## barklike.dog

My personal commerce + creative site - a storefront (clothing) and live radio station. 

Home page features 17 bauhaus-esque geomtric generations (grid, spiral, golden-ratio, albers color studies, towers, morse, etc.) expressed from a curated palette. 

Colors react to the live radio in real time: stream -> web audio analyser -> spectral flux on bass band (adaptive threshold) -> bpm track. 

24/7 live radio self-hosted with AzuraCast stream played in browser.

Shop - products managed in sanity & checkout with stripe


next.js, react, typescript, sanity cms, stripe, azuracast
